# Super Trolley：Ethics Simulator（超级大电车）

一个移动端优先的伦理决策模拟应用。用户通过连续二选一道德困境题目，得到结果导向、规则导向、共情导向或混合型画像。

## 直接预览

无需安装依赖，直接打开：

```text
standalone.html
```

用于上传分享的静态入口在：

```text
publish/index.html
```

## React 版本

安装 Node.js 后运行：

```bash
npm install
npm run dev
```

构建：

```bash
npm run build
```

## 内容说明

题目用于伦理思辨与娱乐，不提供标准答案，也不构成法律、医疗、政治或道德建议。
